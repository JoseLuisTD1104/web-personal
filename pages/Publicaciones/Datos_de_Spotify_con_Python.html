Datos de Spotify con Python

por Jose Luis • 16 abril 2025

Hola a todos. En esta nueva publicación quiero compartir un tema práctico y muy interesante para quienes desean aplicar Python al análisis de datos reales: cómo trabajar con datos de Spotify. Elegí este tema porque permite combinar la programación con el análisis musical, y representa una excelente forma de introducirse al uso de APIs y visualización de datos. Además, Spotify ofrece una API pública muy completa que nos permite acceder a información útil como canciones, artistas, géneros, duración, popularidad y mucho más.

¿Qué es la API de Spotify?

La API de Spotify (Spotify Web API) es una interfaz que permite acceder programáticamente a la información del catálogo de Spotify. Con ella, es posible obtener datos sobre pistas, álbumes, listas de reproducción, artistas, e incluso información de usuarios, como sus canciones favoritas o listas creadas, siempre que se tenga autorización adecuada.

Para usarla, necesitas:

  • Crear una cuenta de desarrollador en Spotify for Developers.
  • Crear una aplicación para obtener tus claves de acceso (client ID y client secret).
  • Usar una biblioteca de Python como spotipy, que simplifica las solicitudes HTTP y el manejo de tokens.

Configuración inicial con Python

Antes de realizar cualquier consulta, es necesario instalar y configurar la biblioteca spotipy:

pip install spotipy

Y luego autenticarte:


import spotipy
from spotipy.oauth2 import SpotifyClientCredentials

sp = spotipy.Spotify(auth_manager=SpotifyClientCredentials(
    client_id="TU_CLIENT_ID",
    client_secret="TU_CLIENT_SECRET"
))

¿Qué tipo de datos se pueden obtener?

Con la API puedes obtener múltiples atributos musicales de una canción, como:

  • name: nombre de la canción
  • artists: artista(s) que la interpretan
  • popularity: nivel de popularidad
  • duration_ms: duración en milisegundos
  • explicit: si contiene lenguaje explícito
  • danceability, energy, valence: métricas sobre su carácter musical

Ejemplo de búsqueda:

track = sp.search(q='track:Blinding Lights artist:The Weeknd', type='track')

¿Cómo se visualiza la información?

Con librerías como pandas, seaborn y matplotlib puedes visualizar relaciones entre datos, como energía y popularidad:


import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Supongamos que ya tienes una lista de canciones
df = pd.DataFrame(canciones)

sns.scatterplot(data=df, x='energy', y='popularity')
plt.title('Relación entre energía y popularidad')
plt.show()
